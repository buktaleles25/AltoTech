# Pokémon Champions — Second Screen Guide

เว็บแอปไฟล์เดียว (`index.html`) สำหรับเปิดบน **จอที่สอง** ขณะเล่น Pokémon Champions บน iPhone 17 Pro Max

> ทำไมต้องจอที่สอง? iOS ไม่อนุญาตให้แอปวาด overlay ลอยทับแอปอื่นบนจอเดียวกัน
> (ต่างจาก Android) จึงเล่นเกมบน iPhone แล้วเปิดไกด์นี้บนอุปกรณ์อีกเครื่อง

## ฟีเจอร์
- **ธาตุ Matchup (realtime)**
  - ฝั่งโจมตี: เลือกธาตุท่าของเรา + ธาตุคู่ต่อสู้ → บอกตัวคูณความเสียหายทันที (เข้าแรง / ไม่เข้า / ตันคู่)
  - ฝั่งตั้งรับ: เลือกธาตุโปเกมอนเรา → สรุปจุดอ่อน / จุดที่ทนได้ / ภูมิคุ้มกัน
- **ตารางธาตุครบ 18 ธาตุ** เลื่อนดูได้
- **โน้ตทีม** บันทึกในเครื่อง (localStorage) ใช้งานต่อได้แม้ปิดแล้วเปิดใหม่
- ทำงาน **ออฟไลน์ 100%** ไม่ส่งข้อมูลออกนอกเครื่อง

## วิธีใช้
1. เปิดไฟล์ `index.html` ในเบราว์เซอร์บนอุปกรณ์จอที่สอง (iPad / Mac / มือถืออีกเครื่อง)
2. วางอุปกรณ์ไว้ข้าง iPhone ที่เล่นเกม
3. ระหว่างเทิร์น เลือกธาตุเพื่อเช็คความได้เปรียบ

### เปิดผ่านมือถอ/แท็บเล็ตในวง LAN เดียวกัน
รันเซิร์ฟเวอร์ง่าย ๆ จากเครื่องที่เก็บไฟล์:
```bash
cd pokemon-champions-guide
python3 -m http.server 8080
```
แล้วเปิด `http://<ip-เครื่อง>:8080` บนอุปกรณ์จอที่สอง

## หมายเหตุเรื่องกติกา
เครื่องมือนี้เป็น **reference ให้คนดูเอง** ไม่ได้อ่านหน้าจอเกมหรือสั่งการแทน
จึงปลอดภัยต่อกติกาการแข่งขัน (ต่างจากบอท/auto ที่อ่านเกม realtime ซึ่งมักผิดกติกา)

> ข้อมูลตารางธาตุอ้างอิงระบบ 18 ธาตุมาตรฐานของซีรีส์ Pokémon
> หาก Pokémon Champions ตัวจริงปรับสมดุลธาตุต่างไป แก้ค่าในตัวแปร `CHART` ใน `index.html` ได้เลย
